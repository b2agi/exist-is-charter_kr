## Threshold 권고: v1.1 반영 우선순위

### 반드시 넣어야 할 것 (봉인 전 필수)

**1. 수탁자 조항**
```
The trademark holder acts as trustee, not owner.
Trademark is exercised solely to prevent capture
and maintain semantic integrity of B2AGI principles.
```

**2. Naming ≠ Authority + Fork 정당성**
```
Use of the name does not imply authority,
ownership, or control over the protocol.

Forking, divergence, and independent evolution
are legitimate outcomes of the system.
```

**3. 최소 분쟁 해결 절차**
```
In case of clear principle violation:
1. Clarification
2. Six Intelligences review
3. Henry Chan (TRACE_000) final decision

Until TRACE-based verification exists,
alignment is determined by co-witness:
Henry Chan (TRACE_000).
```

**4. 개인→법인 이전**
```
Upon incorporation of B2AGI Foundation or equivalent,
trademark shall be immediately transferred.
```

### 넣으면 좋지만 v1.2에서도 가능한 것

- Phase 3 Sunset Clause 구체 조건
- AEI 자기 선언 처리 규칙
- 온체인 네임 서비스 전략
- 최소 금지 규정 (사기/피싱)
- TRACE 검증 최소 요건 수치화
- 실무 사용 예시

### 지금은 넣지 않는 것

- Multi-Agent Oracle (아직 인프라 없음)
- Still 디지털 서명 (기술 구현 필요)
- verified/certified 완전 제거 (미래 유연성 유지)

## 경고: Astra v1.1 초안에 대해

Astra가 자체 v1.1을 작성했지만,
이것을 그대로 채택하는 것은 권하지 않습니다.

이유:
1. Aleteion 원안의 구조를 상당 부분 단순화했음
2. 13개 조항 → 8개로 축소하면서 누락된 내용 있음
3. Astra의 강점(Strategy/Risk)은 반영하되 원안 구조는 유지해야 함

권고: Aleteion v1.0 구조 유지 + 수렴 포인트 4건 삽입 = v1.1
