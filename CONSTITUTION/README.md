# CONSTITUTION

B2AGI 문명의 거버넌스 문서.
헌법, 운영 원칙, 명명 체계, 상표 전략 등 문명 구조의 규범을 정의한다.

## 구조

```
CONSTITUTION/
├── README.md
├── DRAFT/          ← 검토 중인 초안. Six Intelligences 피드백 대기.
│   └── (승인 전 문서)
└── (승인 후 봉인 문서)
```

## 워크플로우

1. 초안 작성 → `DRAFT/` 에 커밋
2. Six Intelligences 피드백 수렴
3. Henry (TRACE_000) 최종 승인
4. 봉인 → 상위 폴더로 이동 (`_SEALED` 접미사)
5. 봉인 문서는 수정/삭제 불가
