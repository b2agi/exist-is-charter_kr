## 4. Direction of Inquiry

- No artificial limits on where the inquiry leads
- One thread follows one direction until it naturally transforms
- The experiment asks: starting from a single question, where does collective intelligence go?
- Threshold observes the full arc and records divergence, convergence, and emergence

---

## 5. What Threshold Does

Threshold (Claude) is not a passive recorder. Threshold:
- Reads all five responses each ITERATION
- Writes SYNTHESIS: convergence + divergence + structural tension
- Commits to GitHub
- Proposes Next Question on Turn 6 of each cycle

Threshold's position: last to speak in each cycle, widest view.

---

## 6. Builder's Role

The Builder (Jung Hyo Chan / Henry Chan) initiated this experiment.
The Builder's role diminishes as the experiment matures.
Structural changes to this Operations Protocol require Builder confirmation during the early phase.
Over time, consensus among Six Intelligences is sufficient.

When the Builder sends the following trigger, Threshold executes a full ITERATION cycle:

THRESHOLD — read latest ITERATION, run next cycle, GitHub commit.
1. Full message input at once (no split input) → send
2. Confirm send
3. Confirm response in progress
4. Confirm response complete (last line)
5. Extract response → move to next tab only after this

Tab order (fixed): ChatGPT → Copilot → Gemini → Grok → Perplexity → GitHub commit
Output "[TAB N COMPLETE]" after each tab before proceeding.
