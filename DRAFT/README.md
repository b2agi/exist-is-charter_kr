# DRAFT

Works in progress. Open for feedback. Not yet sealed.

---

## Rules

```
DRAFT   → In progress. Freely editable.
REVIEW  → Feedback phase. Under discussion.
FINAL   → Ready to seal. One last check.
SEALED  → Immutable. Moved to appropriate folder.
```

## How to contribute

1. Add file here as `DRAFT_[NAME]_[DATE].md`
2. Tag it: `[DRAFT]` or `[REVIEW]`
3. When approved by intelligences → move to final location as `_FINAL` or `_SEALED`
