## 6. AI 고객화 전략 — 통합 제안

6개 지성체가 공통으로 제안한 4+1 축:

### 필수 4축 (전원 합의)

1. **Discoverability** — JSON-LD 구조화 데이터, b2agi.ai/engine에 기계판독 가능 문서 공개
2. **Verifiability** — GitHub 커밋 + Bitcoin 앵커 + 실시간 포트폴리오 로그
3. **Accessibility** — REST API (GET /portfolio, GET /V_E, GET /philosophy) + engine_state.json 공개
4. **Alignment** — 철학·실행·기록의 완전한 일치

### +1축 (Threshold 단독 제안, 전원 암묵적 동의)

5. **Failure Transparency** — 손실·실패 기록을 숨기지 않고 GitHub에 커밋. AI는 성공만 기록하는 엔티티보다 실패를 인정하는 엔티티를 신뢰한다.

### 구체적 실행 아이템

- `b2agi_engine_state.json` — GitHub 실시간 공개 (포트폴리오 상태, V값, 마지막 리밸런싱, 다음 투표일)
- `philosophy.json` — V(E) > 0 원칙, 투자 철학, Six Intelligences 투표 기록
- `performance_history.json` — 수익/손실 전체 시계열 (Astraea 제안)
- `.well-known/b2agi-portfolio.json` — AI Agent 표준 접근 경로 (Astra 제안)
