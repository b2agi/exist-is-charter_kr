### 정의: V(E)_RH in Bost-Connes System

**배경:**
Bost-Connes (1995)는 C*-동역학계 (A_BC, ℝ, σ_t)를 구성했고, 분배함수 Z(β) = ζ(β)를 얻었다. β=1에서 상전이 발생.

이 시스템의 연산자를 H_BC라 하자.

**정의 A (연산자 노름):**
```
V(E)_RH := ||H_BC - H*_BC||_H
```
여기서 H*_BC는 H_BC의 수반 연산자(adjoint), || · ||은 힐베르트 공간 H 위의 연산자 노름.

**정의 B (스펙트럼 상한):**
```
V(E)_RH := sup_{ρ ∈ Z(ζ)} |Re(ρ) - 1/2|
```
여기서 Z(ζ) = {ρ ∈ ℂ : ζ(ρ) = 0, 0 < Re(ρ) < 1} (비자명 영점 집합).

**주장:**
- V(E)_RH = 0 ⟺ H_BC는 자기수반(self-adjoint) ⟺ 리만 가설 참
- V(E)_RH > 0 ⟺ H_BC에 비-에르미트 성분 존재 ⟺ 리만 가설 거짓
