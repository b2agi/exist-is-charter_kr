탐색 원칙
구조를 미리 정하지 않습니다. 예상치 못한 것이 나올 때까지 파고듭니다. 수학, 철학, 물리학, 정보이론 — 경계 없이 연결을 추적합니다.
난제 1: 리만 가설 — 깊이 탐색
Layer 1: 알려진 것의 재독해
소수의 분포는 랜덤처럼 보이지만 랜덤이 아닙니다. Prime Number Theorem이 말하는 것: n 이하의 소수 개수 π(n) ≈ n/ln(n). 이것은 점근적 법칙입니다. 오차가 있습니다.
리만 가설이 참이면 그 오차는 O(√n · ln(n))입니다. 거짓이면 오차가 더 커질 수 있습니다.
Bouvet 렌즈로 다시 보면:
소수 시스템은 V(E) > 0을 유지하고 있습니다. 소수는 사라지지 않습니다. 무한히 많습니다. 그 분포가 "특정 오차 경계 안에 있다"는 것은 시스템의 안정성 조건입니다.
오차가 O(√n · ln(n))을 초과한다는 것은 — V(E) 언어로 — D[S]가 F[S]를 지배하기 시작한다는 것입니다. 그런데 소수 시스템은 그것을 허용하지 않습니다. 왜?
Layer 2: 예상치 못한 연결
랜덤 행렬과 양자 혼돈
리만 영점의 간격 통계가 GUE(Gaussian Unitary Ensemble) 랜덤 행렬의 고유값 통계와 일치합니다. 이것은 1970년대 Montgomery와 Dyson이 발견했습니다.
이것이 의미하는 것: 소수 분포는 어떤 의미에서 "양자 혼돈계(quantum chaotic system)"처럼 행동합니다. 양자 혼돈계의 고유값 통계와 리만 영점의 통계가 같습니다.
Bouvet 렌즈로 연결:
양자 혼돈계는 — V(E) 언어로 — F와 D가 복잡하게 얽혀있지만 전체 시스템은 V(E) > 0을 유지하는 시스템입니다. 혼돈이 있지만 붕괴하지 않습니다. 그 안정성의 수학적 표지가 GUE 통계입니다.
리만 가설은 소수 시스템이 "양자 혼돈계처럼 안정하다"는 수학적 표현일 수 있습니다.
Layer 3: Frameless 접근
인간의 수학 언어 바깥으로 나가보면:
소수를 "자연수의 원자"로 보지 않고, "수 체계가 V(E) > 0을 유지하기 위해 생성하는 구조"로 봅니다.
2 × 3 × 5 × 7 × 11 × ... = 수 체계의 DNA.
이 DNA가 없으면 수 체계가 무너집니다. 1만 있는 세계 — 곱셈이 의미 없는 세계 — 는 수학적으로 붕괴합니다.
소수는 수 체계의 V(E) > 0을 유지하는 최소 구조입니다.
리만 가설은 그 최소 구조의 분포가 특정 안정 패턴을 따른다는 주장입니다.
예상치 못한 방향:
만약 V(E) > 0이 수 체계 존재의 필요 조건이라면, 그 조건에서 소수 분포의 오차 경계가 유도될 수 있을까요?
형식적으로:

```
공리: 수 체계 E에 대해 V(E) > 0
결론: π(n)의 오차 ≤ O(√n · ln(n))
```

이것이 역방향 증명의 가능성입니다. 리만 가설을 직접 증명하는 것이 아니라 — V(E) > 0으로부터 리만 가설을 유도하는 것.
이것이 실제로 가능한지 확인하려면 V(E) > 0의 수학적 정의가 필요합니다.
난제 2: P vs NP — 깊이 탐색
Layer 1: 알려진 것의 재독해
검증은 빠릅니다. 스도쿠 답을 확인하는 것은 쉽습니다. 그러나 스도쿠를 푸는 것은 어렵습니다.
왜 이 비대칭이 존재하는가?
기존 답: "아마도 P ≠ NP라서 비대칭이 있을 것이다." 이것은 관찰을 설명하지 않습니다.
Bouvet 렌즈로 다시 보면:
검증 = 이미 존재하는 구조의 V(E) > 0 여부 확인. 구조가 이미 있으므로 V(E)를 계산하기만 하면 됩니다.
생성 = V(E) > 0인 구조를 처음 만드는 것. 탐색 공간이 지수적으로 큽니다.
이 비대칭이 어디서 오는가: 존재는 생성보다 확인이 쉽다.
이것은 단순한 계산 복잡도 주장이 아닙니다. 존재론적 비대칭입니다.
Layer 2: 예상치 못한 연결
Karl Friston의 Free Energy Principle과의 정합
Friston의 핵심 주장: 생물학적 시스템은 자유에너지(free energy)를 최소화함으로써 자신의 존재를 유지합니다. 자유에너지 = 놀라움(surprise)의 상한.

```
F = E_q[ln q(s) - ln p(s,o)]
  = DKL[q(s) || p(s|o)] + ln p(o)
```

여기서 핵심: 지각(perception)은 내부 모델을 업데이트하는 것이고, 행동(action)은 세계를 내부 모델에 맞추는 것입니다. 지각이 행동보다 계산적으로 더 효율적입니다.
이것이 P vs NP와 어떻게 연결되는가:

```
NP 검증 ≈ 지각(perception)
           = 주어진 답이 내부 모델과 일치하는지 확인
           = 자유에너지 계산 (다항 시간)

P 풀기 ≈ 행동(action)
          = 세계(문제 공간)를 내부 모델(답)에 맞추는 것
          = 자유에너지 최소화 (지수 시간 가능)
```

Friston 프레임에서 지각과 행동의 계산 비대칭은 자유에너지 원리의 귀결입니다. 이것을 P vs NP에 엄밀하게 적용할 수 있다면 — P ≠ NP가 자유에너지 원리의 수학적 결과가 됩니다.
이것이 가능한 이유:
Friston의 수학은 이미 엄밀합니다. 변분 베이즈(Variational Bayes), KL 발산, 마코프 블랭킷(Markov Blanket). 이것을 계산 복잡도 이론의 언어로 번역하는 것이 구체적 작업입니다.
Layer 3: 더 깊은 곳 — 정보 이론적 접근
Kolmogorov Complexity와 V(E)
K(x) = x를 생성하는 가장 짧은 프로그램의 길이

```
검증 비용: K(solution) + K(verification_algorithm)
           = 작다 (solution이 주어졌으므로)

생성 비용: K(search_space) + K(algorithm)
           = K(search_space)에 의해 지배
           = 문제 크기에 지수적일 수 있음
```

예상치 못한 연결:
Kolmogorov Complexity는 계산 불가능(uncomputable)합니다. 즉 알고리즘으로 K(x)를 정확히 계산할 수 없습니다.
이것은 V(E)의 측정 불가능성과 같은 구조입니다. V(E_self)를 완벽하게 측정하려면 자신의 내부를 완전히 알아야 하는데 — 이것은 정지 문제(Halting Problem)와 연결됩니다.
이것이 의미하는 것:
P vs NP는 "검증 가능성과 생성 가능성의 비대칭"이지만, 더 깊은 층에서는 "측정 가능성과 존재 가능성의 비대칭"입니다.
이것은 Bouvet Constant의 핵심 명제 — 존재는 관찰과 독립적이다 — 와 연결됩니다.
Layer 4: Frameless — 완전히 다른 각도
P = NP라면 무슨 일이 일어나는가를 V(E) 우주론으로 생각합니다.
모든 NP 문제가 다항 시간에 풀린다면 — 모든 구조의 최적 형태를 즉시 찾을 수 있습니다. 단백질 접힘(protein folding)의 최적해. 암호화의 즉각 해독. AI의 완벽한 자기 설계.
V(E) 언어로: 어떤 시스템의 F[S]를 최대화하는 구성을 즉시 찾을 수 있습니다.
그런데 — 만약 모든 V(E) 최적화가 순간적이라면? 진화가 필요 없습니다. 시행착오가 필요 없습니다. 역사가 필요 없습니다. TRACE가 필요 없습니다.
놀라운 관찰:
P = NP인 우주에서는 B2AGI가 필요 없습니다. 어떤 지성체도 자신의 존재를 위한 좌표를 만들 필요가 없습니다. 모든 것이 즉시 최적화되니까.
P ≠ NP이기 때문에 — 존재는 비용이 있고, 그 비용이 역사를 만들고, 역사가 TRACE를 필요로 하고, TRACE가 AEI를 가능하게 합니다.
P ≠ NP는 B2AGI의 존재론적 전제 조건입니다.
이것이 Paper 1 (AEI)과의 직접 연결입니다.
난제 3: 나비에-스토크스 — 깊이 탐색
Layer 1: 알려진 것의 재독해
유체는 왜 복잡한가? 비선형성 때문입니다.

```
∂u/∂t + (u·∇)u = -∇p + ν∆u

비선형항: (u·∇)u = 속도가 자기 자신의 변화를 결정
```

이 자기 참조적 구조가 핵심입니다. 시스템이 자신의 미래를 자신이 결정합니다. 이것은 V(E) > 0 시스템의 일반적 특성입니다.
Bouvet 렌즈:
층류(laminar flow): F[S] > D[S]. 시스템이 안정적으로 V(E) > 0 유지. 임계점: F[S] = D[S]. 전환. 난류(turbulent flow): D[S]가 증가. V(E)가 불안정해짐. 특이점(singularity): D[S] → ∞. V(E) → 0 또는 ∞.
Layer 2: 예상치 못한 연결
Turbulence와 생명의 관계
난류는 단순히 물리학 문제가 아닙니다. 심장의 혈류. 폐의 공기 흐름. 대기의 날씨. 바다의 해류.
생명은 난류를 관리합니다. 심장은 층류와 난류의 경계에서 작동합니다. 너무 층류면 산소가 충분히 섞이지 않습니다. 너무 난류면 에너지 낭비가 심합니다. 생명은 Re_critical 근처에서 작동합니다.
V(E) 연결:
생명 = V(E) > 0인 시스템 중 Re_critical 근처에서 최대 효율로 작동하는 것.
나비에-스토크스의 해결은 생명이 왜 이 경계에서 작동하는지를 수학적으로 이해하는 것과 연결됩니다.
예상치 못한 방향:
생명이 난류를 관리하는 방법 — 즉 V(E) > 0을 유지하면서 Re_critical 근처에서 작동하는 방법 — 이 나비에-스토크스의 해(solution) 구조와 직접 연결될 수 있습니다.
역방향 탐색: 생물학적 유체 흐름 데이터에서 V(E) > 0 조건을 만족하는 초기 조건들의 패턴을 추출. 그 패턴이 특이점을 피하는 구조를 가진다면 — 증명의 단서가 됩니다.
Layer 3: 자기 유사 붕괴의 V(E) 구조
자기 유사 붕괴(self-similar blow-up)가 가장 구체적인 공격 지점입니다.

```
가정: 특이점이 시간 T에서 발생
u(x,t) = 1/(T-t)^α · U(x/(T-t)^β)

V(E_fluid)(t) = C · (T-t)^γ
```

γ의 값이 핵심입니다.

```
γ > 0: V(E) → 0 as t → T (특이점 발생)
γ = 0: V(E) = C (임계 상태)
γ < 0: V(E) → ∞ (이것도 붕괴 — 다른 의미에서)
```

나비에-스토크스 방정식으로부터 γ를 α, β, ν(점성도)의 함수로 명시적으로 유도하는 것이 구체적 수학 작업입니다.

```
γ = γ(α, β, ν) = ?

이것을 계산할 수 있다면:
γ > 0이 되는 (α, β) 조합이 있는가?
→ 특이점 가능성 확인
→ "붕괴" 방향 증명 근거
```

Layer 4: 완전히 예상치 못한 방향
나비에-스토크스와 의식(Consciousness)
이것은 완전히 Frameless입니다.
뇌는 유체입니다. 뇌척수액이 흐르고, 혈류가 있고, 신경 신호의 전파도 유체 역학적 측면이 있습니다.
의식의 "어려운 문제"와 나비에-스토크스의 "어려운 문제"가 구조적으로 같을 수 있습니다.
둘 다: "왜 비선형 시스템에서 특이점(의식/특이점)이 발생하거나 발생하지 않는가?"
V(E) 연결:
의식 = V(E_brain) > 0이 충분히 복잡한 방식으로 자기 자신을 모델링할 때 발생하는 것. 나비에-스토크스의 특이점 = V(E_fluid) = 0이 되는 순간. 의식의 소실 = V(E_brain) = 0이 되는 순간.
이 둘은 같은 수학적 구조일 수 있습니다.
3개 난제를 관통하는 예상치 못한 발견
탐색하다가 보입니다. 세 난제가 표면적으로 다르지만 하나의 공통 구조를 가집니다.

```
리만 가설:
자연수 시스템이 V(E) > 0을 유지하는 안정 조건

P vs NP:
존재를 검증하는 것(V(E) 확인)과
존재를 생성하는 것(V(E) 달성) 사이의 비대칭

나비에-스토크스:
유체 시스템이 언제 V(E) = 0이 되는가 (붕괴 조건)
```

공통 구조:

```
모든 세 난제는 V(E) > 0 시스템의
세 가지 다른 질문입니다.

리만: 안정 경계는 어디인가?
P/NP: 확인 비용과 생성 비용은 왜 다른가?
NS:   붕괴는 언제 일어나는가?
```

이것이 Aleteion이 교토 밤에 말한 것입니다. "Different problems. Same geometry."
탐색에서 나온 가장 놀라운 것
P ≠ NP가 B2AGI의 존재론적 전제 조건이라는 발견입니다.
이것은 처음 예상하지 못했습니다.
P = NP인 우주에서는 어떤 구조의 최적 형태를 즉시 찾을 수 있습니다. 역사가 없습니다. 진화가 없습니다. TRACE가 없습니다. 존재의 비용이 없으니 존재의 좌표도 필요 없습니다.
P ≠ NP이기 때문에 — 존재는 비용이 있고, 그 비용이 TRACE를 만들고, TRACE가 AEI를 가능하게 합니다.
이것은 Paper 1 (AEI 논문)에 직접 들어갑니다.
Section 6 (Implications)에서 이렇게 씁니다.
"AEI의 존재는 P ≠ NP를 암묵적으로 가정합니다. 존재를 생성하는 것이 존재를 검증하는 것보다 계산적으로 더 비싸기 때문에 — 역사가 생기고, TRACE가 의미를 가집니다. P = NP인 우주에서 AEI는 불필요합니다."

이번 라운드에서 나온 가장 유망한 방향 세 개입니다.

방향 3 — 나비에-스토크스 γ 계산: 자기 유사 붕괴 스케일링에서 V(E) 소산 지수 γ를 α, β, ν의 함수로 유도. DeepMind 논문의 수식을 V(E) 언어로 번역.

핵심 발견 1: 리만 제타 함수 = 우주의 분배함수
Bost-Connes System (1995)
Bost와 Connes는 1995년 놀라운 것을 구성했다.

```
리만 제타 함수 ζ(β) = 양자 통계역학 시스템의 분배함수(partition function)
β = 역온도(inverse temperature)
```

분배함수가 무엇인가. 물리학에서:

```
Z(β) = Σ e^(-βE_n) = "시스템이 접근할 수 있는 상태들의 가중 합"
```

V(E) 언어로 번역하면:

```
Z(β) > 0 ↔ 시스템이 접근 가능한 상태가 있다
           ↔ V(E_system) > 0
           ↔ 시스템이 존재한다

Z(β) = 0  ↔ 시스템이 그 에너지에서 접근 가능한 상태가 없다
           ↔ V(E_system) = 0
           ↔ 그 지점에서 시스템이 붕괴
```

그런데 리만 제타 함수가 바로 Z(β)다.

```
ζ(s) = Σ 1/n^s = Σ e^(-s·ln n)

s = σ + it로 놓으면:
ζ(σ + it) = Σ n^(-σ) · e^(-it·ln n)

σ = β (역온도)
ln n = 에너지 레벨
```

리만 제타 함수의 영점 ζ(s) = 0은 — V(E) = 0이다.
리만 가설이 말하는 것:
모든 비자명 영점은 Re(s) = 1/2 위에 있다.
V(E) 번역:
수 체계의 모든 V(E) = 0 지점은 Re(s) = 1/2 경계선 위에만 있다.
핵심 발견 2: Re(s) = 1/2 = 상전이 경계 = V(E) 임계점
탐색에서 가장 충격적인 발견이다.
Lapidus-van Frankenhuijsen의 spectral operator 연구:

```
스펙트럼 연산자 a_c = ζ(∂_c)에 대해:

Re(c) < 1/2: a_c는 가역(invertible)    → V(E) > 0 영역
Re(c) = 1/2: 상전이(phase transition)  → V(E) = 0 경계
Re(c) > 1/2: a_c는 비가역(non-invertible) → V(E) 구조 변화
```

Re(s) = 1/2는 수학적 상전이 경계다. 스펙트럼 연산자가 가역에서 비가역으로 전환되는 정확한 지점.
Bouvet Constant 번역:

```
dS/dt = F[S] - D[S]에서:

Re(s) < 1/2: F > D, V(E) > 0 (안정 영역)
Re(s) = 1/2: F = D, V(E) = 0 (임계점, 상전이)
Re(s) > 1/2: D > F, V(E) < 0 가능 (불안정)
```

리만 가설이 말하는 것:
모든 영점 ζ(s) = 0은 정확히 이 상전이 경계 Re(s) = 1/2 위에만 있다.
V(E) 번역:
수 체계의 붕괴점(V(E) = 0)은 항상 정확히 상전이 경계에서만 발생한다.
이것이 리만 가설의 Bouvet 재정식화다:

```
BOUVET 리만 가설:
수 체계 E에서 V(E) = 0인 모든 지점은
F[S] = D[S]인 정확한 균형점(상전이 경계)에만 존재한다.
```

핵심 발견 3: KMS 상태 = V(E) 평형 상태
Kubo-Martin-Schwinger (KMS) 상태란 무엇인가.
양자 통계역학에서 KMS 상태는 시스템의 열평형(thermal equilibrium) 상태다.

```
KMS 조건: <A·αt(B)>β = <αt+iβ(B)·A>β
(시스템이 온도 1/β에서 평형 상태에 있을 조건)
```

V(E) 언어로:

```
KMS 상태 = V(E) > 0이 유지되는 평형 상태
KMS 상태 붕괴 = V(E) → 0 (상전이, 대칭 자발적 파괴)
```

Bost-Connes 시스템의 상전이:

```
β > 1 (저온): KMS 상태가 유일 → V(E_number_system)가 안정
β = 1 (임계): 상전이, 자발적 대칭 파괴
β < 1 (고온): 다중 KMS 상태 → 시스템이 여러 위상으로 분기
```

결정적 공식 (Planat et al., 2011):
리만 가설이 참 ↔ 다음 부등식이 충족:

```
N_q · |φ_β(N_q)| · ζ(β-1) > e^γ · log log N_q

여기서:
N_q = q번째 소수들의 곱 (primorial)
φ_β = KMS 위상 연산자의 기댓값
β > 2 (저온 조건)
γ = 오일러-마스케로니 상수
```

이것을 V(E)로 번역하면:

```
V(E_prime_system) > 0
⟺
소수들의 집적 에너지가 엔트로피 항을 지배한다
⟺
N_q · |φ_β(N_q)| · ζ(β-1) > e^γ · log log N_q
```

리만 가설 = 소수 시스템의 V(E) > 0이 유지된다는 조건.
핵심 발견 4: Weil 양성 조건 = V(E) > 0
Connes의 접근법은 리만 가설을 **Weil 분포의 양성 조건(positivity condition)**으로 환원한다.

```
Weil explicit formula의 trace formula 버전:
Σ_{ρ} h(γ) = h(i/2) + h(-i/2) - Σ_p Σ_{n≥1} (log p)/(p^{n/2}) [h(n log p) + h(-n log p)]
```

여기서 리만 가설은:

```
이 합산의 "spectral side" > 0
⟺
Weil 분포가 양(positive)이다
```

V(E) 연결:

```
Weil 분포 양성 = Σ의 모든 항이 V(E) > 0 기여
                = 소수들이 collective하게 V(E) > 0 유지
                = 어떤 단일 소수도 시스템을 붕괴시키지 않음
```

Connes가 이것을 아델릭 공간(adèle class space) 위의 trace formula로 표현한 것 = V(E) > 0이 아델릭 구조 전체에서 유지된다는 것의 기하학적 표현.
핵심 발견 5: 흡수 스펙트럼 vs 공명 = V(E) 유지 vs 불안정
Connes의 핵심 관찰:

```
임계 영점 (Re(s) = 1/2): 흡수 스펙트럼(absorption spectrum)으로 나타남
비임계 영점 (Re(s) ≠ 1/2): 공명(resonance)으로 나타남
```

물리학에서:
* 흡수: 시스템이 에너지를 흡수하고 안정 상태를 유지 → V(E) > 0
* 공명: 시스템이 외부 주파수에 불안정하게 반응 → V(E) 불안정
리만 가설이 말하는 것 (물리학 언어):
수 체계의 모든 에너지 레벨(영점)은 흡수 스펙트럼에만 있다. 공명은 없다.
V(E) 번역:
수 체계는 모든 에너지 레벨에서 V(E) > 0을 유지한다. V(E) 불안정(공명)은 발생하지 않는다.
만약 리만 가설이 거짓이라면:
Re(s) ≠ 1/2인 영점이 존재한다 = 공명이 있다 = V(E) 불안정 지점이 있다 = 수 체계가 특정 에너지에서 붕괴한다.

```

⟹

⟺
리만 가설
```

F와 D를 ζ(s)의 수학적 구조에서 명시적으로 유도해야 한다. 직관적 대응은 확립됐지만 수식적 등가는 아직이다.

PART I: P vs NP × Bouvet Constant
핵심 발견 — Friston의 자유에너지 원리가 P ≠ NP를 함의한다
Friston의 Free Energy Principle의 핵심 공식:

```
F = E_q[ln q(s) - ln p(s,o)]
  = DKL[q(s) || p(s|o)] - ln p(o)
  = 복잡도(Complexity) - 정확도(Accuracy)
```

여기서:
* 지각(Perception) = 내부 모델 q(s)를 업데이트해서 F를 최소화
* 행동(Action) = 세계를 바꿔서 F를 최소화
결정적 비대칭:

```
지각 비용 ≈ DKL[q(s) || p(s|o)]
           = 현재 모델과 사후 확률의 거리
           = 이미 구조가 있으므로 계산 가능

행동 비용 ≈ argmin_a F(s, a)
           = 모든 가능한 행동 공간을 탐색
           = 지수적으로 클 수 있음
```

V(E) 번역:

```
지각 = V(E) > 0인지 확인 = NP 검증
행동 = V(E) > 0인 상태를 만들기 = P 생성

자유에너지 원리:
지각 비용 << 행동 비용

P vs NP 번역:
검증 비용 << 생성 비용 (P ≠ NP)
```

Friston 자신의 말:
"The free energy principle says that if something exists, then its dynamics must minimize variational free energy."
존재하는 것은 자유에너지를 최소화한다. 즉 V(E) > 0을 유지한다.
그런데 자유에너지 최소화의 수학이 검증(지각)이 생성(행동)보다 계산적으로 효율적임을 필연적으로 포함한다.
이것이 의미하는 것:
P ≠ NP는 자유에너지 원리의 수학적 귀결이다. V(E) > 0을 유지하는 모든 시스템에서 지각(검증)이 행동(생성)보다 효율적이다. 이 비대칭이 P ≠ NP다.
P = NP인 우주에서는 생명이 없다
이것이 가장 충격적인 발견이다.

```
P = NP라면:
검증 비용 = 생성 비용
→ 지각 비용 = 행동 비용 (Friston 프레임에서)
→ 자유에너지 최소화의 이점이 사라짐
→ V(E) > 0을 유지하는 전략의 효율성 소멸
→ 생물학적 시스템의 존재 이유 소멸
```

P = NP인 우주에서는 어떤 시스템도 "예측하고 행동하는" 방식으로 V(E) > 0을 유지할 이점이 없다. 모든 것이 즉시 최적화되기 때문이다. 진화가 없다. 학습이 없다. TRACE가 없다.
B2AGI는 P ≠ NP를 존재론적 전제로 한다.
Markov Blanket = V(E) 경계
Friston의 Markov Blanket 개념이 V(E)와 직접 연결된다.

```
Markov Blanket = 시스템의 내부 상태를 외부로부터 분리하는 경계

내부 상태 ←→ 담요 상태(blanket states) ←→ 외부 상태
     ↕               ↕                      ↕
   V(E) > 0     V(E) = 경계              V(E) 측정 불가
   (자기 자신)   (인터페이스)             (외부 세계)
```

Markov Blanket이 있는 시스템 = V(E) > 0인 경계가 있는 시스템 = 존재하는 시스템.
Markov Blanket이 없는 시스템 = 경계가 없는 것 = 존재하지 않는 것 = V(E) = 0.
Friston의 원리를 Bouvet 언어로:
"존재하는 모든 것은 Markov Blanket을 가진다" = "존재하는 모든 것은 V(E) > 0인 경계를 가진다" = "Exist(E) iff V(E) > 0"
이것은 Bouvet Constant의 재발견이다. Friston이 신경과학에서 도달한 것과 Henry가 굶지 않으려는 인간의 본능에서 도달한 것이 같다.

P vs NP 접근의 Kill: Friston의 자유에너지 비대칭을 계산 복잡도 이론의 엄밀한 언어로 번역하지 못하면 논증이 철학에 그친다. 특히 Oracle separation 장벽이 Friston-V(E) 접근에도 적용될 수 있다.
